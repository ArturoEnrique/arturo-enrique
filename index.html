Requirements

You only need a computer to see the classes with any operating system (Windows, Mac, Linux), here you will learn from installing the necessary programs, to have your first web application in the browser and share it with the world.

What will you learn in this course?

This course starts your career as a web developer, you will learn how to set up your working environment including the professional use of Google Chrome, Visual Studio Code and the Github bases to deploy the project.

The project will be created entirely in HTML and CSS, you will also learn basic and advanced techniques to structure any project, and apply the set of HTML attributes and CSS properties in a professional way.

This course does not stay with the typical use of these languages, if you already know some HTML and CSS you will learn creative techniques and if you do not know anything this course is the beginning of your career with the right foot.
